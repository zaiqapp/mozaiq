# Mozaiq Roadmap

## V1 — Open Source MVP ✅
- Drag & drop canvas with react-grid-layout
- 11 widget types with realistic mock data
- 3 starter templates (Analytics, Inventory, Purchasing)
- AI natural language dashboard generator
- Save dashboard to database
- Shareable read-only view
- Embeddable via iframe
- One-click Deploy to Vercel

## V2 — Auth + Real Data Sources ✅
- NextAuth.js (GitHub + Google)
- User accounts + /my-dashboards
- Google Sheets connector
- CSV upload with column mapper
- AI chart recommendations
- AI dashboard insights (Pro)
- AI template customization

## V3 — Billing & SaaS Tiers
- Stripe Checkout
- Free / Pro ($15/mo) / White-label ($199/mo)
- Feature gating by plan

## V4 — Teams & More Connectors
- Team workspaces + RBAC
- REST API connector
- Embeddable JS snippet

## V5 — White-label & Enterprise
- Custom domain (CNAME + SSL)
- Full brand removal
- Direct DB connectors (Postgres, MySQL)
